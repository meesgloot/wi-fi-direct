курсовая работа 

# Задание
Реализовать программму с помощью которой передаются некоторые данные по реальной сети
Второй тип работ дополниетельные требования 
1. Описание цели и постановки задачи;
2. Описание реализуемой системы;
3. Описать список ограничений введенных при разработке и реализации;
4. Описание программных и аппаратаных средств;
5. Описание работы реализованной программы в виде псевдокода или блок-схемы;
6. Описание выполненного эксперемента;
7. Определение характеристик реализованной системы;
8. Пример работы системы при некоторых зафиксированных условиях;
9. Выводы по проделанной работе
